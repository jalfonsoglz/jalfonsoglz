## 👋 Hola, soy Alfonso González. Consultor funcional y desarrollador de Odoo ERP, ayudo a pequeñas y medianas empresas a optimizar sus procesos con **implementaciones, automatizaciones y desarrollos personalizados**, donde el objetivo de cada proyecto debe **mejorar productividad**, **reducir errores** y **generar valor real**.

## 🧠 Experiencia
- Implementación de módulos Odoo: **CRM, Ventas, Compras, Punto de Venta, Inventario, Contabilidad, Proyectos, Mesa de Ayuda, Flota, Servicio Externo, Citas, Mantenimieno, Odoo Studio.**.  
- Desarrollo de módulos a medida para flujos de negocio específicos.  
- Configuración avanzada, capacitación de usuarios y soporte post-implementación.  
- Integración de automatizaciones y dashboards para análisis de KPI y control operativo.
